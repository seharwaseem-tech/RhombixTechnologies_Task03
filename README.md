# RhombixTechnologies_Task03
This repository contains a Personal Library Management System created as part of the internship task.

The system is developed using HTML, CSS, and JavaScript.
It is designed to manage books digitally with features like adding, borrowing, returning, searching, and deleting books in a clean and user-friendly interface.

## Project Overview

This library management system includes:

- Add new books with title, author, category, and cover image
- Borrow and return book functionality
- Late return fine calculation
- Borrow history tracking
- Search books by title or author
- Filter books by category
- Delete books option
- Local Storage support to save data


## Technologies Used

- HTML5
- CSS3
- JavaScript (ES6)
- Browser Local Storage


## Purpose of This Task

The purpose of this task is to demonstrate:

- Understanding of HTML structure
- CSS styling and layout techniques
- JavaScript DOM manipulation
- Dynamic content rendering
- Local storage data management
- Basic logic implementation (borrow system & fine calculation)
